# Club Pokémon
Crea una aplicación para revisar si una persona puede o no entrar al club pokémon:

## condiciones:
Si la persona tiene al menos 7 medallas y es socio del club, puede entrar
Si la persona no es socio, no puede entrar
Si la persona no tiene 7 medallas, no puede entrar
Si la persona tiene 8 medallas, puede pasar a ser socio Elite (grupo elite)

## Uso del DOM
Para que la app, sea más usable, necesitamos manejo de DOM. Para este caso, vamos a disponer de un arreglo de personas en nuestro código JS y en HTML, disponibilizarlos a travez de una lista (o select), para que al momento de seleccionar a una persona, sepamos si tiene permitido o no el acceso a nuestro club.
